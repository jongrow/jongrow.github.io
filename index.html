外汇交易不是说你一天24小时盯盘就一定会赚钱的，可能你这24小时中绝大部分都是无效的。真正的行情大部分都是在关键时间点，在调整充分之后，在极短的时间内快速走完。实际交易中，关注仅有的一部分高效时间就足够了。
严格的风控 + 极致的交易系统 + 资金管理 + 执行力，三者却一不可，必备技能点全部点亮 + 无数次的重复练习，形成潜意识的规则，

发表于 2016-09-05   |   分类于 外汇   |  

投资是时间的函数，也是个漫漫的长跑。因此并不是要你跑得如何快，而是如何跑完全程。我们经常看到在一段时间内出现盈利很高的投资人，巴菲特那个50年23％的投资收益水平，对很多人来说并不算什么。但全球做到这个时间、这个成绩的人似乎只有这一个。因此，如果你在3个月挣到了500％，并不能说一定成功，重要的是在”长跑”中同样可以有稳定的收益。想做到这个稳定的收益，必须要遵守一个原则：一定要控制亏损！当有规律和这个原则相违背的时候，遵守控制亏损的原则。

2016年9月5日欧磅

价格跌破水平支撑位之后，水平方向震荡调整到均线15EMA之后受压继续走弱，均线15EMA,50SMA空头流向，关注 0.8375 向下跌破的交易。

2016年8月25黄金

周图

前面几周连续冲高回落，收出带上影线的蜡烛形态，本周目前是跌破水平位，均线15EMA,50SMA多头流向，短线偏空

天图

昨天收出坚决阴线，跌破水平支撑位，均线15EMA拐头向下，短线偏空

4小时图

坚决大阴线跌破水平支撑带，奠定空头基调，均线15EMA,50SMA空头流向，关注调整之后向下跌破的交易机会

1小时图

一波下跌之后，调整到15EMA之后，受压走弱，值得注意的是均线15EMA,50SMA开口较大。

15分钟图

开仓
合理的地方
  1. 一波下跌之后，水平方向震荡调整到1小时图15EMA之后，受压走弱
  2. 破位时1小时图收坚决阴线，顶在水平支撑位附近
  3. 风险可控，最大止损40个点，最小止损放起跌处25个点，盈亏比 1:2
  4. 水平方向震荡调整之后，行情开始启动
不合理地方
  1. 破位是15分钟均线50SMA是走平的
  2. 缺乏明显的诱空动作
平仓

  1. 理论上如果行情走弱的话，是不应该再去测试上方1327的位置，如果再次测试触发止损离场
  2. 15分钟向上测试拐点压力线及15EMA，如果再度走弱创出新低的话，所有单子全部保护起来
  3. 一波下跌之后，行情受到天图拐点支撑线的支撑展开回调，测试5分钟图15EMA之后，再次有走弱的可能，此时最正确的做法是
    将移动止损到次低点
  4. 最终行情没法持续走弱，再次展开更大幅度的回调，保护离场
回顾

在已经有40多点盈利的情况，行情展开回调没法持续走弱，应该先平仓一部分，剩下的一部分继续持有看看
